# Introduction
"_***Windows Server*** is a brand name for a group of server operating systems released by [Microsoft](/Tech-Ref/Microsoft) since 2003. The first Windows server edition to be released under that brand was _Windows Server 2003_. However, the first server edition of Windows was _Windows NT 3.1 Advanced Server_, followed by_ Windows NT 3.5 Server_, _Windows NT 3.51_ Server, _Windows NT 4.0 Server_, and _Windows 2000 Server_. _Windows 2000 Server_ was the first server edition to include [Active Directory](/Tech-Ref/Software-Development/DevOps-\(Development-and-IT-Operations\)/AD-\(Active-Directory\)), [DNS Server](NewPage), [DHCP Server](NewPage), [Group Policy](NewPage), as well as many other popular features used today._"
