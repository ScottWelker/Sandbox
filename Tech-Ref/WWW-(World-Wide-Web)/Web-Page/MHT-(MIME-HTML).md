# Introduction
"_A ***[MIME]() [HTML](/Tech-Ref/WWW-\(World-Wide-Web\)/HTML-\(Hypertext-Markup-Language\)) (MHTML)*** file is a [webpage](/Tech-Ref/WWW-\(World-Wide-Web\)/Web-Page) archive saved by a [web browser](/Tech-Ref/WWW-\(World-Wide-Web\)/Web-Browser), such as Google Chrome or Microsoft Edge. It is saved in the [MIME](/Tech-Ref/WWW-\(World-Wide-Web\)/Web-Page/MHT-\(MIME-HTML\)) [HTML](/Tech-Ref/WWW-\(World-Wide-Web\)/HTML-\(Hypertext-Markup-Language\)) or ***MHTML*** format, which stores [HTML](/Tech-Ref/WWW-\(World-Wide-Web\)/HTML-\(Hypertext-Markup-Language\)), [CSS](/Tech-Ref/WWW-\(World-Wide-Web\)/CSS-\(Cascading-Style-Sheets\)), [JavaScripts](/Tech-Ref/Software-Development/JavaScript), images, and other linked resources into a single file. ***MHT*** files are more commonly saved as .MHTML files._"
