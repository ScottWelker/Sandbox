# Introduction
***remark-lint-no-undefined-references*** is a [unified](/Tech-Ref/Software-Development/JavaScript/Node.js/unified) ([remark](/Tech-Ref/Software-Development/JavaScript/Node.js/unified/remark)) plugin, specifically a [remark-lint](/Tech-Ref/Software-Development/JavaScript/Node.js/unified/remark/remark%2Dlint) rule. Lint rules check markdown code style.

This specific "_Lint rule_" is noteworthy because it is used, with customization, to validate (_lint_) [this Wiki's](/Tech-Ref/Microsoft/Microsoft-Azure/ADO-\(Azure-DevOps\)/Wiki-\(Azure-DevOps\)) [markdown](/Tech-Ref/Software-Development/Markup-Language/Markdown) files. See [remark-cli, Linting THIS Wiki](/Tech-Ref/Software-Development/JavaScript/Node.js/unified/remark/remark%2Dcli#linting-this-wiki).
