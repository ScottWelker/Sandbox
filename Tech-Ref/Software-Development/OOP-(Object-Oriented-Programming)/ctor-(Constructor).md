# Introduction
"_In class-based [object-oriented programming](/Tech-Ref/Software-Development/OOP-\(Object-Oriented-Programming\)), a ***Constructor (ctor)*** is a special type of subroutine called to create an object. It prepares the new object for use, often accepting arguments that the constructor uses to set required member variables.._"
